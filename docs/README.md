# Documentation

This folder contains documentation for changes, improvements, and modifications made to the ForzaETH Race Stack.

## File Naming Convention

Use the format: `YYMMDD-title.md`

Example: `251123-overtaking-enhancement.md`

## Current Documentation

- **251123-overtaking-enhancement.md** - Context-aware autonomous overtaking system (original proposal)
- **251125-package-structure.md** - Package 2 initial structure and message definitions
- **251125-implementation-spec.md** - Scoped implementation plan with simplified approach
- **TEAM.md** - Team member information and package ownership

## Adding New Documentation

When documenting changes:
1. Create a new markdown file with the YYMMDD-title.md format
2. Include: objective, architecture, implementation details, and status
3. Keep it concise and technical
4. Link related issues or pull requests if applicable
